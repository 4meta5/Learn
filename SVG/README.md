SVG diff
=====
This skeleton is treated very differently than a typical programing language; as it is actually a markup language.

Primitives/Shapes:
* ellipse
* circle
* line
* path
* image
* Gradient (radial)
* text

Attributes:
* x,y
* height
* width
* fill
* stroke
* stroke-width
* id
* opacity
* fill opacity
* stroke-linecap

Modifiers:
* transform-rotate
* transform-translate
* transform-scale
* Grouping
* Use
